# Walmart Sales Forecasting

This project focuses on predicting Walmart weekly sales using machine learning techniques based on historical data.

## Objective

To forecast weekly sales using:
- Data preprocessing
- Feature engineering
- Machine learning models
- Performance evaluation

## Dataset Files

- train.csv
- test.csv
- features.csv
- stores.csv

The dataset includes historical sales data, store information, and external features.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- scikit-learn
- SciPy
- Jupyter Notebook
